#plus_one
What comes after x? plus_one knows!*
 
( * only up to 999)
##Install
Download it, I guess?

##Use
###Singular
```python
plus_one(1) # returns 2
plus_one('two') # returns 'three'
plus_one('THREE') # returns 'FOUR'
```

###For Loop
```python
number = 'zero'
while number != 'five':
    number = plus_one(number)
    print(number)

# returns 'one two three four' 
```
